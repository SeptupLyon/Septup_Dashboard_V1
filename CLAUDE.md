# Project

Simple HTML/JavaScript app with Airtable API and Vercel deployment.

```
api/airtable.js   — Airtable API calls
api/generate.js   — generation logic
index.html        — frontend (HTML + JS)
vercel.json       — deployment config
```

# Rules

- Minimal changes only — touch only what is needed for the task
- No refactoring unless explicitly asked
- No new files unless strictly necessary
- No comments unless logic is non-obvious
- No error handling for impossible scenarios

# Working style

- One feature at a time
- Read the file before editing it
- Prefer editing existing code over adding new code
- Ask before doing anything destructive or irreversible
